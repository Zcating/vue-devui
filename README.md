# Vue DevUI

## 安装依赖

```
yarn(推荐)

or

npm i
```

### 安装单个依赖包

```
yarn add dayjs
```

### 安装开发依赖包

```
yarn add -D @commitlint/cli @commitlint/config-conventional husky
```

### 移除依赖包

```
yarn remove dayjs
```

## 启动

```
yarn dev(推荐)

or

npx vite

or

npm run dev
```

## 访问

[http://localhost:3000/](http://localhost:3000/)

## 生产打包

```
yarn build(推荐)

or

npm run build
```
